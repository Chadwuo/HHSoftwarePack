### 入门指南
HHSoftwarePack 是基于NSIS方式的安装包制作工具，界面可以自由定制，拿来即用。非常适合为windows程序快速制作安装可执行setup.exe文件

下载Releases中最新安装包，即刻体验全部功能，你无需涉足各种脚本工具，只需面对你所需要打包的程序本身，让一切你所需要的精美安装界面效果，从这里信手拈来。

### 程序文件目录说明
eg:

```
HHSoftwarePack.exe
├── /Output/ //输出目录
├── /x64/    //64位所需dll
├── /x86/    //32位所需dll
├── /SoftSetupCore/  //核心打包库
├── SoftwarePack.exe //主执行程序

```
